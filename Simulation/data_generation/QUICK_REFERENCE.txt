QUERY BALANCE SHEETS
--------------------
SELECT household_id, netWorth, primaryHomeValue, liquidSavings
FROM world_sim_simulations.household_balance_sheet_samples
WHERE simulation_id = 'your-sim-id'
  AND sim_clock_datetime = (
      SELECT simulation_start_datetime 
      FROM world_sim_simulations.simulations 
      WHERE simulation_id = 'your-sim-id'
  );

KEY CONCEPTS
------------
• Modeling unit: household (not individual agent)
• One balance sheet per household at simulation start
• Idempotent: safe to call multiple times
• Deterministic: same household → same balance sheet
• Automatic: integrated into bulk agent initialization


HOUSEHOLD RESOLUTION
--------------------
Priority:
1. Residence_Families_FamilyID
2. Mailing_Families_FamilyID  
3. SYNTH_{LALVOTERID}


HPI RESOLUTION
--------------
Priority:
1. CBSA (if ConsumerData_CBSA_Code exists)
2. MSA (if ConsumerData_MSA_Code exists)
3. STATE (from Residence_Addresses_State)

COMMON QUERIES
--------------

1. Get balance sheet for specific agent:
   
   SELECT bs.*
   FROM world_sim_simulations.initialized_agents ia
   JOIN world_sim_agents.l2_other_part_1 o1 ON o1.LALVOTERID = ia.agent_id
   JOIN world_sim_simulations.household_balance_sheet_samples bs
     ON bs.household_id = COALESCE(
         o1.Residence_Families_FamilyID,
         o1.Mailing_Families_FamilyID,
         CONCAT('SYNTH_', ia.agent_id)
     )
   WHERE ia.agent_id = 'your-agent-id'
     AND ia.simulation_id = 'your-sim-id';

2. Get households with net worth > $500k:
   
   SELECT household_id, netWorth, primaryHomeValue
   FROM world_sim_simulations.household_balance_sheet_samples
   WHERE simulation_id = 'your-sim-id'
     AND netWorth > 500000
   ORDER BY netWorth DESC;

3. Get average balance sheet by net worth bucket:
   
   SELECT 
       net_worth_bucket,
       COUNT(*) as count,
       AVG(netWorth) as avg_net_worth,
       AVG(primaryHomeValue) as avg_home_value,
       AVG(liquidSavings) as avg_savings
   FROM world_sim_simulations.household_balance_sheet_samples
   WHERE simulation_id = 'your-sim-id'
   GROUP BY net_worth_bucket;
